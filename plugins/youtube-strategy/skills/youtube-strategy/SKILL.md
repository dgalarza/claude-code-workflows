# YouTube Strategy Skill

This skill helps streamline the content creation workflow for YouTube videos by generating essential elements that drive engagement and discovery. It assists with crafting compelling titles that balance SEO and curiosity, writing detailed descriptions that inform viewers and boost searchability, creating attention-grabbing hooks for the first 10-15 seconds of video, suggesting thumbnail concepts that stand out in feeds, and drafting pinned comments that encourage community interaction.
Use this skill when you're preparing to publish a video and need help with any of these content elements. The skill takes into account YouTube's algorithm preferences, audience psychology, and best practices for technical content while maintaining an authentic voice that resonates with developers and AI engineering audiences.

This Skill is broken up into multiple resources per topic. Reference each when you are solving a particular problem.

**Description** `resources/description.md`
**The Hook** `resources/hook.md`
**Pinned Comment** `resources/pinned-comment.md`
